## ‚öôÔ∏è How To Set Up

### 1. Rename the `.env.example` file to `.env`

```bash
mv .env.example .env
```

Now open the `.env` file and add your discord bot `TOKEN` to the file.

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

OR

```bash
pipenv install
```

### 3. Start the bot

Linux

```bash
python3 main.py
```
Windows

```ps
python main.py
```
